					Git Commands

git status 						-> prints the stauts of the Git repository
git init 						-> initializes the Git Repository
git config --global user.name "Lets's Kode It" 		-> Specify Username globally
git config --global user.email "letskodeit@gmail.com" 	-> Specify Email globally
git config user.name "Lets's Kode It" 			-> Specify Username for particular repository
git config --user.email "letskodeit@gmail.com" 		-> Specify Email for particular repository
git add 						-> Add file to staging environment
git commit -m "" 					-> Commit file to local Git repository
git log 						-> Check log history
git diff <filename> 					-> Check changes made to file with existing file
git remote add origin<GitHub Repository URL> 		-> Establish connection to the remote GitHub
git push origin master 					-> Push to master branch
